## 当前生成方式

当前项目使用内置固定海报引擎直接生成正式海报，不需要再去第三方平台搭模板。

你只需要准备：

- 合并后的固定背景图
- 每位学员的录取相关图片
- 标题、副标题和三阶段升学路径文案

系统会在项目内部自动完成固定排版，并根据 `1~3` 张图片切换对应的内置布局。
